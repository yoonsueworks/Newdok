# 내게 필요한 뉴스레터, 빠르게 찾을 수 있는 방법 없을까? 📨 

## 🍎 프로젝트 소개

> 여러 어떤 브랜드를 구독해야 하는지 모르겠거나,
> 뉴스레터가 일반 메일과 섞여 정작 구독한 뉴스레터를 읽지 못할 때
> 사용자에게 필요한 뉴스레터를 맞춤 추천하고 구독을 관리해주는 서비스입니다.
